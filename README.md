# Mycologist

Welcome to Mycologist! This is a management video game where you play as a mycologist who owns a mushroom shop. Your goal is to find mushrooms in the forest, create various mushroom products, and sell them to customers.

## Getting Started

To run the game locally, follow these steps:

1. Clone this repository to your local machine.

    ```bash
    git clone https://github.com/electricganesha/mycologist.git

    ```

2. Navigate to the project directory.

    ```bash
    cd mycologist

    ```

3. Install dependencies.

    ```bash
     npm install

    ```

4. Start the development server.

    ```bash
     npm install
    ```

5. Open your browser and navigate to http://localhost:3000 to play the game.
